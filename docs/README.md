## 👋 Por onde começar

Se você acabou de entrar no projeto, siga esta ordem:

**1.** 📋 **Visão Geral do Produto** → entenda o que estamos construindo, para quem e por quê

**2.** 🛠️ **Considerações Técnicas** → stack, bounded contexts, ADRs e decisões arquiteturais

**3.** 📌 **Regras de Negócio Transversais** → regras que valem para todos os módulos

**4.** Mergulhe no módulo em que vai atuar (seções abaixo)

---
