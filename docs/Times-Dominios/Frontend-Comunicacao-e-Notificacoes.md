# 🎨 Frontend — Comunicação & Notificações

## Microfrontend Apps

| App | Rota base |
| --- | --- |
| `app-chat` | `/messages/*` |
| `app-notifications` | `/notifications/*` |

## Telas

| Rota | Tela |
| --- | --- |
| `/messages` | Lista de conversões |
| `/messages/:conversation_id` | Chat em tempo real (WebSocket) |
| `/notifications` | Central de notificações in-app |
| `/settings/notifications` | Preferências de notificação por canal |
