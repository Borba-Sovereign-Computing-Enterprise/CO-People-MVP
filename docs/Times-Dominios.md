# 🏙️ Times & Domínios

> Esta é a área de **ownership por domínio**. Cada ilha é responsável por um conjunto coeso de funcionalidades — com seus próprios microsserviços, microfrontends, infra e roadmap de entrega.
> 

---

## Como funciona

- Cada ilha é **autônoma**: tem seu backend, frontend e infra mapeados
- Cada ilha tem um **Roadmap + Kanban próprio** para gerenciar entregas
- O responsável pela ilha é o **DRI** (Directly Responsible Individual) de tudo que acontece nela
- Atividades são criadas no Kanban, com data de início, entrega e finalização
