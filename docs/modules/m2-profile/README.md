
# Módulo 2 — Perfil do Profissional

> **Plataforma de Saúde** · Documentação Técnica v1.0 · Março 2026 · Confidencial

O perfil é a vitrine do profissional dentro da plataforma. Deve transmitir confiança ao paciente por meio de informações verificadas, avaliações e histórico.

---

## 3.1 Seções do Perfil

Informações exibidas ao cliente no perfil do profissional:

| Seção | Descrição |
|---|---|
| Foto e nome | Identificação visual do profissional |
| Especialidade(s) e subespecialidades | Áreas de atuação |
| Formação acadêmica | Graduação, pós, residência, especializações |
| Número de registro profissional | Ex: CRM/SP 123456 |
| Anos de experiência e áreas de foco | — |
| Modalidade de atendimento | Presencial, domiciliar ou ambas |
| Valor da consulta | Por modalidade |
| Disponibilidade de horários | Prévia |
| Avaliações e comentários | De pacientes anteriores |

---

## 3.2 Sistema de Avaliação

Após cada consulta concluída, o cliente é convidado a avaliar o profissional com **nota de 1 a 5 estrelas** e comentário textual opcional.

- Avaliações ficam públicas no perfil após **moderação**
- O profissional pode **responder** à avaliação publicamente
- Avaliações só são permitidas para consultas **efetivamente realizadas**
- A nota média é exibida com o **número total de avaliações**
- Denúncias de avaliações inadequadas são analisadas pelo **administrador**

---

## 3.3 Modalidade de Atendimento

| Modalidade | Regras específicas |
|---|---|
| Presencial | Exige endereço do consultório cadastrado e validado |
| Domiciliar | Exige raio de atendimento e taxa adicional configurável |
| Ambos | Cliente escolhe a modalidade na hora do agendamento |
